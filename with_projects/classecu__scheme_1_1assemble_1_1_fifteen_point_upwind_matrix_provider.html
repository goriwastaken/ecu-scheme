Detailed Description

template<typename SCALAR, typename FUNCTOR>
class ecu_scheme::assemble::FifteenPointUpwindMatrixProvider< SCALAR, FUNCTOR >

Class providing the element matrix for the 15-point upwind scheme The element matrix provider evaluates the convective bilinear form for the 15-point upwind scheme \( b_h^{full}(u_h,v_h) \) presented in section 2.1.1 of the thesis.
