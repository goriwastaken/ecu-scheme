Detailed Description

template<typename SCALAR, typename FUNCTOR>
class ecu_scheme::assemble::StableConvectionUpwindMatrixProvider< SCALAR, FUNCTOR >

Class providing the element matrix for the 7-point upwind scheme The element matrix provider evaluates the convective bilinear form for the 7-point upwind scheme \( b_h^{vmb}(u_h,v_h) \) presented in section 2.1.1 of the thesis.

Template Parameters
SCALARthe scalar type of the FE space
FUNCTORthe type of the velocity field \(\vec{\beta}\)
