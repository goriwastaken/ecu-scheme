Detailed Description

template<typename SCALAR, typename FUNCTOR>
class ecu_scheme::assemble::EdgeElementMassMatrixProvider< SCALAR, FUNCTOR >

Class providing the edge element mass matrix for a given FE space and velocity field \(\vec{\beta}\). The element matrix provider evaluates part of the bilinear form for 1-forms from section 3.1, namely:

\[ (\omega_h, \eta_h) \mapsto \int_T \mathcal{I}_{\vec{\beta}, 1}^1(\mathrm{i}_{\vec{\beta}} \mathrm{d}^1 \omega_h) \wedge \star \eta_h \]

where the basis functions for \(\omega_h, \ \eta_h \) are the edge element basis functions defined as:

\[ \mathbf{\psi}_i = s_i (\lambda_i \mathbf{grad}_{\Gamma}(\lambda_{i+1}) - \lambda_{i+1} \mathbf{grad}_{\Gamma}(\lambda_{i})) \]

Template Parameters
SCALARthe scalar type of the FE space
FUNCTORthe type of the velocity field \(\vec{\beta}\)
